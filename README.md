# MKSU-susfs
此版本包含MKSU-SKN测试版 和 SUSFS v1.5.5

功能：
-> MKSU-SKN测试版
->解决Holmes异常环境
-> SUSFS ඞ v1.5.5
-> 手动Hook，实现更好隐藏的
-> Wireguard支持
->用于LineageOS检测的简单Maphide
->用于jit-zygote-cache检测的无用Maphide
-> Magic Mount支持
-> 5.10-6.1以上的BBRv3

注意事项：
-> 由于非 kprobe 挂钩，SUS SU 模式 2 将显示为已取消或不兼容，并且不再需要！
